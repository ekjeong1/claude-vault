# 마르코프 체인

고객 행동의 전이 확률을 모델링하는 확률론적 도구.

## 핵심 개념

**마르코프 체인**은 현재 상태만으로 다음 상태를 예측하는 확률 모델. "과거는 현재에 이미 반영되어 있다"는 전제.

## 비즈니스 적용

- 고객 여정 분석: 신규 → 활성 → 이탈 전이 확률
- 구독 모델: 유지율, 업그레이드, 다운그레이드 예측
- 재고 관리: 품절 → 보충 → 과잉 상태 전환

## 예시

신규 고객이 한 달 뒤 활성 상태일 확률 60%, 이탈 확률 40%. 3개월 뒤 상태 분포를 행렬 곱셈으로 계산.

## 효과

장기 고객 가치(LTV) 예측, 개입 시점 파악, 리소스 배분 최적화
