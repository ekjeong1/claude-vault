# 극값 이론

## 개념

극값 이론(Extreme Value Theory, EVT)은 확률 분포의 꼬리(tail) 부분, 즉 극단적 사건의 발생 가능성과 영향을 분석하는 통계학 분야다.

## 핵심 원리

**분포의 꼬리가 중요한 이유**
- 평균과 표준편차만으로는 극단적 손실 포착 불가
- 희귀하지만 치명적인 사건(Black Swan)
- 정규분포 가정의 한계

## 주요 분포

**GEV(Generalized Extreme Value) 분포**
```
극값 = 블록 최댓값의 분포
```

**GPD(Generalized Pareto Distribution)**
```
임계값 초과 분포
```

**형태 매개변수 ξ**
- ξ > 0: Heavy tail (무한 모멘트)
- ξ = 0: Exponential tail
- ξ < 0: Short tail (유한 상한)

## Tail Risk 관리

**VaR(Value at Risk)**
- 특정 신뢰수준에서 최대 예상 손실

**CVaR(Conditional VaR, Expected Shortfall)**
- VaR 초과 시 평균 손실
- VaR보다 리스크를 보수적으로 평가

## 수학적 정의

**일변수 함수의 극값 조건**:

1. **1차 필요조건**: $f'(x^*) = 0$

2. **2차 충분조건**:
   - $f''(x^*) > 0$ → 극소(local minimum)
   - $f''(x^*) < 0$ → 극대(local maximum)

**다변수 함수** $f: \mathbb{R}^n \rightarrow \mathbb{R}$:

1. **1차 조건**: $\nabla f(x^*) = 0$

2. **2차 조건** (헤시안 행렬 $H$):
   $$H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}$$
   - 모든 고유값 $> 0$ → 극소
   - 모든 고유값 $< 0$ → 극대
   - 고유값이 양/음 혼재 → 안장점(saddle point)

**제약 조건하 최적화** (라그랑주 승수법):
$$\mathcal{L}(x, \lambda) = f(x) - \sum_{i=1}^{m} \lambda_i g_i(x)$$

$$\nabla_x \mathcal{L} = 0, \quad g_i(x) = 0 \quad \forall i$$
## 실무 적용

**금융 리스크 관리**
- 시장 급락 시나리오 분석
- 자본 적정성 평가
- 스트레스 테스트

**보험 및 재난 관리**
- 대형 재해 보험료 산정
- 극한 기후 사건 대비

**운영 리스크**
- 시스템 장애의 극단적 영향
- 공급망 중단 리스크

## 의사결정 관점

- **꼬리 리스크 인식**: 평균만 보지 말고 극단 시나리오 고려
- **보수적 전략**: 극단 손실 회피를 우선시
- **헤징과 보험**: 극값 발생 시 보호 장치
