# 엔트로피

## 개념

엔트로피(Entropy)는 데이터의 불확실성 또는 무질서도를 측정하는 정보 이론의 핵심 개념이다.

## 정의

Shannon 엔트로피:
```
H(X) = -Σ p(x) log₂ p(x)
```

- 확률 분포가 균등할수록 엔트로피가 높다
- 확실성이 높을수록 엔트로피가 낮다

## 의사결정에서의 활용

**정보 가치 평가**
- 정보 획득으로 인한 엔트로피 감소 = 정보 가치
- 의사결정 트리에서 최적 분기 선택

**신호 대 잡음 비율**
- 높은 엔트로피 = 높은 잡음
- 낮은 엔트로피 = 명확한 신호

**불확실성 정량화**
- 예측 모델의 신뢰도 평가
- 리스크 수준 측정

## 수학적 정의

**섀넌 엔트로피(Shannon Entropy)**:

$$H(X) = -\sum_{i=1}^{n} p_i \log_2 p_i$$

여기서:
- $X$: 확률변수
- $p_i = P(X = x_i)$: 사건 $x_i$의 확률
- 단위: bits (log base 2)

**연속 확률분포**:
$$h(X) = -\int_{-\infty}^{\infty} f(x) \log f(x) \, dx$$

**조건부 엔트로피**:
$$H(Y|X) = -\sum_{x,y} p(x,y) \log \frac{p(x,y)}{p(x)}$$

**상호 정보량(Mutual Information)**:
$$I(X;Y) = H(X) + H(Y) - H(X,Y)$$

**볼츠만 엔트로피** (통계역학):
$$S = k_B \ln \Omega$$
($\Omega$: 미시상태 수, $k_B$: 볼츠만 상수)

## 응용 분야

- 머신러닝: 의사결정 트리, 특성 선택
- 통신: 데이터 압축, 채널 용량
- 투자: 포트폴리오 다양화 평가
