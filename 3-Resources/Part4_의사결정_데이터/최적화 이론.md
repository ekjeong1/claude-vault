# 최적화 이론

## 개념

최적화 이론은 제약 조건 하에서 목적 함수를 최대화하거나 최소화하는 방법론이다.

## 핵심 요소

**목적 함수(Objective Function)**
- 최대화하려는 가치 (수익, 효용)
- 최소화하려는 비용 (시간, 자원)

**제약 조건(Constraints)**
- 예산, 시간, 물리적 한계
- 정책적, 법적 제약

## 선형계획법(Linear Programming)

```
Maximize:   c₁x₁ + c₂x₂ + ... + cₙxₙ
Subject to: a₁₁x₁ + a₁₂x₂ + ... ≤ b₁
            a₂₁x₁ + a₂₂x₂ + ... ≤ b₂
            x₁, x₂, ... ≥ 0
```

**적용 사례**
- 자원 배분 (인력, 예산, 재고)
- 생산 계획 최적화
- 포트폴리오 구성

## 비선형 최적화

- 경사 하강법(Gradient Descent)
- 라그랑주 승수법(Lagrange Multipliers)
- 유전 알고리즘, 시뮬레이티드 어닐링

## 실무 적용

- **공급망**: 물류 경로, 재고 수준
- **금융**: 자산 배분, 리스크 관리
- **마케팅**: 광고 예산 배분
- **운영**: 스케줄링, 작업 할당
