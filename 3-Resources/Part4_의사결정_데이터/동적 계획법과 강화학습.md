# 동적 계획법과 강화학습

## 동적 계획법(Dynamic Programming)

순차적 의사결정 문제를 부분 문제로 분해하여 최적해를 구하는 방법론.

**핵심 원리**
- **벨만 방정식**: 최적 정책의 재귀적 구조
- **부분 문제 최적성**: 전체 최적해는 부분 최적해로 구성

**활용 예시**
```
V(s) = max[R(s,a) + γ·V(s')]
```
- s: 현재 상태
- a: 행동
- R: 보상
- γ: 할인율
- s': 다음 상태

## 강화학습(Reinforcement Learning)

환경과 상호작용하며 시행착오를 통해 최적 정책을 학습하는 적응적 방법론.

**주요 구성 요소**
- **에이전트(Agent)**: 의사결정 주체
- **환경(Environment)**: 상태 변화 시스템
- **보상(Reward)**: 행동의 피드백

**핵심 알고리즘**
- Q-Learning: 가치 함수 학습
- Policy Gradient: 정책 직접 최적화
- Actor-Critic: 두 방법의 결합

## 실무 적용

**순차적 의사결정**
- 투자 전략: 시간에 따른 포트폴리오 조정
- 재고 관리: 주문 시점과 수량 결정

**적응적 시스템**
- 개인화 추천: 사용자 피드백 반영
- 가격 최적화: 수요 반응 학습
- 자율 시스템: 로봇, 자율주행

## 장점

- 불확실한 환경에서 학습 가능
- 장기적 결과 고려
- 환경 변화에 적응
