# 의사결정과 데이터 분석

데이터 기반 의사결정을 위한 핵심 수학적 프레임워크 모음.

**↑ [[../0_BrainTwin_Master_Index|BrainTwin 마스터 인덱스로 돌아가기]]**

---

## 개요

이 섹션은 불확실성 하에서 최적의 의사결정을 내리기 위한 6가지 핵심 이론을 다룬다:

- **인과 관계 규명**: 진짜 원인을 어떻게 찾는가?
- **실험과 학습**: 어떻게 테스트하면서 최적화할 것인가?
- **불확실성 측정**: 어떻게 불확실성을 정량화할 것인가?
- **최적화**: 제약 조건 하에서 최선의 선택은?
- **순차적 의사결정**: 시간에 따라 어떻게 적응할 것인가?
- **극단 리스크**: 최악의 시나리오를 어떻게 대비할 것인가?

---

## 핵심 개념

### [[인과 추론]]
**진짜 원인을 찾는 과학**

- 상관관계 vs 인과관계 구분
- 혼재 변수 통제
- 무작위 실험과 준실험 설계
- 활용: A/B 테스트, 정책 효과 측정, 전략 검증

### [[실험과 학습]]
**학습하면서 최적화하기**

- A/B 테스트: 통제된 실험
- Multi-Armed Bandit: 탐색하면서 활용
- 탐색-활용 트레이드오프
- 베이지안 최적화: 최소 시도로 최적값 찾기
- 활용: 제품 최적화, 추천 시스템, 마케팅 캠페인

### [[엔트로피]]
**불확실성의 측정과 정보 가치**

- Shannon 엔트로피로 데이터의 불확실성 정량화
- 정보 획득의 가치 평가
- 신호와 잡음 구분
- 활용: 의사결정 트리, 특성 선택, 리스크 측정

### [[최적화 이론]]
**제약 조건 하 자원 배분**

- 목적 함수 최대화/최소화
- 선형계획법과 비선형 최적화
- 실무 적용: 포트폴리오 구성, 예산 배분, 스케줄링
- 활용: 자원 배분, 생산 계획, 투자 전략

### [[동적 계획법과 강화학습]]
**순차적·적응적 의사결정**

- 벨만 방정식과 최적 정책
- 시행착오를 통한 학습
- 불확실한 환경에서의 적응
- 활용: 투자 전략, 재고 관리, 개인화 시스템

### [[극값 이론]]
**Tail Risk와 극단 시나리오**

- 확률 분포의 꼬리 분석
- VaR/CVaR로 극단 손실 측정
- Black Swan 사건 대비
- 활용: 금융 리스크 관리, 재난 대비, 스트레스 테스트

## 이론 간 관계

```
인과 관계 규명 ─────────┐
  (인과 추론)           │
                       ▼
               실험과 학습 ──────────┐
            (A/B 테스트, MAB)       │
                                   ▼
                          불확실성 측정 ─────────┐
                            (엔트로피)           │
                                                ▼
                                         의사결정 최적화
                                                │
                  ┌─────────────────────────────┼─────────────────────┐
                  ▼                             ▼                     ▼
             순차적 최적화                 제약 조건 최적화        극단 시나리오 대비
            (동적 계획법)                  (최적화 이론)            (극값 이론)
```

**통합 관점:**
- [[인과 추론]]으로 진짜 원인을 밝히고
- [[실험과 학습]]으로 테스트하면서 최적 전략을 찾고
- [[엔트로피]]로 불확실성을 측정하고
- [[최적화 이론]]으로 최선의 배분을 찾고
- [[동적 계획법과 강화학습]]으로 시간에 따라 적응하며
- [[극값 이론]]으로 극단 리스크를 관리한다

## 실무 통합 사례

### 마케팅 캠페인 효과 측정
1. [[인과 추론]] - 캠페인의 진짜 효과 규명
2. [[실험과 학습]] - A/B 테스트와 MAB로 채널별 성과 비교
3. [[엔트로피]] - 고객 세그먼트 불확실성 평가
4. [[최적화 이론]] - 예산을 채널별로 최적 배분
5. [[동적 계획법과 강화학습]] - 실시간 성과 기반 예산 재배분
6. [[극값 이론]] - 최악의 ROI 시나리오 대비

### 포트폴리오 관리
1. [[인과 추론]] - 자산 수익률 결정 요인 분석
2. [[실험과 학습]] - 새로운 자산군 소량 투자로 탐색
3. [[엔트로피]] - 자산 다양화 평가
4. [[최적화 이론]] - 평균-분산 최적화로 자산 배분
5. [[동적 계획법과 강화학습]] - 시장 변화에 따른 리밸런싱
6. [[극값 이론]] - VaR/CVaR로 최대 손실 제한

### 공급망 최적화
1. [[인과 추론]] - 공급 중단 원인 규명
2. [[실험과 학습]] - 신규 공급업체 소량 주문으로 품질 테스트
3. [[엔트로피]] - 수요 불확실성 측정
4. [[최적화 이론]] - 재고 수준과 물류 경로 최적화
5. [[동적 계획법과 강화학습]] - 수요 패턴 학습과 적응
6. [[극값 이론]] - 공급 중단 등 극단 시나리오 대비
