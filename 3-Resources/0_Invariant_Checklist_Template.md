## 개요

이 템플릿은 BrainTwin에 **새로운 개념, 도구, 방법론을 추가**할 때 5가지 불변량을 자동으로 검증합니다.

**사용 시기:**
- 새 개념 파일 작성 시

- 외부 프레임워크 통합 시

**규칙:**
- **최소 20/25점 (80%)** 필요
- 하나라도 0점이면 재설계
- 검증 결과를 [[0_Concept_Validation]]에 기록

## 불변량 1: 복잡성 단순화

**원칙:** "모든 개념은 본질만 남기고 나머지는 제거한다"

### 질문

#### Q1. 압축성 (Compression)

**질문:** 이 도구가 복잡성을 줄이는가?

- [ ] **5점** - 10개 이상 변수를 2-3개 핵심 요인으로 압축
- [ ] **4점** - 5-10개 변수를 3-4개로 압축
- [ ] **3점** - 복잡성 유지 (변화 없음)
- [ ] **2점** - 복잡성 약간 증가
- [ ] **1점** - 복잡성 크게 증가
- [ ] **0점** - 복잡성이 문제 해결보다 큼

**점수:** ___/5

**근거:**


**예시:**
- ✅ SVD/PCA: 30개 KPI → 2-3개 요인 (5점)
- ❌ 5단계 다차원 회귀 분석 (1점)

---

#### Q2. 직관성 (Intuitiveness)

**질문:** 비전공자도 5분 안에 이해 가능한가?

- [ ] **5점** - 수학 배경 없이 즉시 이해, 시각화로 설명 가능
- [ ] **4점** - 간단한 예시로 5-10분 내 이해
- [ ] **3점** - 기초 수학 지식 필요, 20-30분 학습
- [ ] **2점** - 전공 지식 필요, 1시간+ 학습
- [ ] **1점** - 전문가만 이해 가능
- [ ] **0점** - 수학이 목적이 됨 (비즈니스 의미 불명확)

**점수:** ___/5

**근거:**


**예시:**
- ✅ 그래프 이론: 화살표 그림으로 직관적 (5점)
- ❌ "Hilbert 공간의 orthogonal projection" (0점)

---

#### Q3. 즉시성 (Immediacy)

**질문:** 현장에서 즉시 적용 가능한가?

- [ ] **5점** - 별도 학습 없이 바로 사용 가능
- [ ] **4점** - 10-20분 준비 후 적용
- [ ] **3점** - 1시간 내 준비 후 적용
- [ ] **2점** - 반나절 이상 준비 필요
- [ ] **1점** - 며칠 학습 후 가능
- [ ] **0점** - 전문가 도움 필수

**점수:** ___/5

**근거:**


---

#### Q4. 제거 가능성 (Necessity)

**질문:** 이 도구 없이는 문제 해결이 불가능한가?

- [ ] **5점** - 이 도구만이 유일한 솔루션
- [ ] **4점** - 대안보다 10배 효율적
- [ ] **3점** - 대안보다 2-3배 효율적
- [ ] **2점** - 대안과 비슷한 효율
- [ ] **1점** - 대안이 더 나음
- [ ] **0점** - 이 도구가 필요 없음
