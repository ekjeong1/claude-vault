## 개요

조합적 창의성은 **완전히 새로운 것을 만드는 것이 아니라**, 기존 요소들을 **새로운 방식으로 결합**하여 혁신을 창출하는 접근법이다. 대부분의 혁신은 "무에서 유"가 아니라 "조합의 재배치"다.

### 핵심 통찰

**"새로운 아이디어는 존재하지 않는다. 새로운 조합만이 존재한다."**

- iPhone = 전화 + iPod + 인터넷 + 터치스크린
- Uber = 택시 + GPS + 모바일 결제 + 피드백 시스템
- Airbnb = 숙박 + P2P 거래 + 리뷰 + 보험

## 핵심 개념

### 1. 순열과 조합 (Permutations and Combinations)

**수학적 기초**:

**순열 (Permutation)**: 순서가 중요한 배열
```
P(n,r) = n! / (n-r)!
```

**조합 (Combination)**: 순서가 무관한 선택
```
C(n,r) = n! / (r!(n-r)!)
```

**비즈니스 적용**:

| 요소 수 (n) | 2개 조합 수 C(n,2) | 3개 조합 수 C(n,3) |
|------------|-------------------|-------------------|
| 5 | 10 | 10 |
| 10 | 45 | 120 |
| 20 | 190 | 1,140 |
| 100 | 4,950 | 161,700 |

**통찰**: 요소가 많을수록 가능한 조합은 **기하급수적 증가**.

**예시: 레고 블록**
- 6개의 2x4 레고 블록 = 915,103,765가지 조합
- 단순한 요소 + 조합 규칙 = 무한한 창조

### 2. SCAMPER (창의적 사고 체크리스트)

**정의**: 기존 제품/서비스를 체계적으로 변형하는 7가지 방법.

| 기법 | 질문 | 예시 |
|------|------|------|
| **S**ubstitute (대체) | 무엇을 다른 것으로 바꿀 수 있나? | 플라스틱 → 생분해성 소재 |
| **C**ombine (결합) | 무엇과 결합할 수 있나? | 카메라 + 전화 = 스마트폰 |
| **A**dapt (적응) | 다른 맥락에 어떻게 적용? | 군사 GPS → 민간 내비게이션 |
| **M**odify (수정) | 크기/형태를 어떻게 바꿀까? | 노트북 → 태블릿 → 폴더블 |
| **P**ut to other uses (전용) | 다른 용도로 사용? | 베이킹소다 → 탈취제 |
| **E**liminate (제거) | 무엇을 없앨 수 있나? | 유선 → 무선 (케이블 제거) |
| **R**everse (역전) | 순서/역할을 뒤집으면? | 교사 → 학생 (플립 러닝) |

**체계적 적용**:
```
기존 제품 → SCAMPER 7가지 적용 → 7개 변형 아이디어
→ 조합 (C(7,2) = 21개) → 추가 아이디어
```

### 3. 인접 가능성 (Adjacent Possible)

**정의**: 현재 상태에서 **한 걸음 떨어진 혁신**만 실현 가능하다.

**생물학적 비유**:
- 진화는 점진적: 날개는 하루아침에 생기지 않음
- 중간 형태가 있어야 함: 앞다리 → 날개 원형 → 날개

**비즈니스 적용**:

```
          [현재 기술/시장]
               ↓
    ┌─────────┼─────────┐
    ↓         ↓         ↓
 [인접 1]  [인접 2]  [인접 3]  ← 실현 가능
    ↓         ↓         ↓
 [인접 1.1] [인접 2.1] ...     ← 다음 단계
    ↓
  [미래]                       ← 여러 단계 후
```

**핵심**:
- 너무 멀리 도약하면 실패 (시장/기술 준비 안 됨)
- 인접한 것부터 차례로 실현 → 새로운 인접 가능성 열림

**예시: 스마트폰 진화**
```
피처폰 (2000)
  → PDA + 전화 (BlackBerry, 2003)
  → 터치 + 앱 스토어 (iPhone, 2007)
  → AI 비서 (Siri, 2011)
  → AR/VR 통합 (Vision Pro, 2024)
```

각 단계는 이전 단계의 인접 가능성.

## Part 5 내 연결

### [[TRIZ와 Isomorphism]]

**관계**: TRIZ 40가지 원리 = 검증된 조합 패턴.

- TRIZ는 "이미 해결된 문제"의 패턴 카탈로그
- 조합적 창의성은 이 패턴들을 재조합
- 예: "분할" + "역전" = 모듈화된 역방향 프로세스

### [[비유와 유추]]

**관계**: 비유 = 영역 간 조합.

- 유추는 "A 영역의 요소 + B 영역의 문제" 조합
- 개념 블렌딩 = 두 영역의 창의적 조합
- 예: "바이러스 전파" + "마케팅" = "바이럴 마케팅"

### [[제약 기반 혁신]]

**관계**: 제약은 조합 공간을 제한.

- 제약 = 가능한 조합의 부분집합으로 제한
- MVP = 최소 조합으로 시작
- TOC = 병목이라는 제약 하에서 최적 조합 찾기

### [[Chaos와 Fractal]]

**관계**: 프랙탈 = 자기 조합의 반복.

- 프랙탈: 같은 패턴의 재귀적 조합
- 카오스: 단순 규칙의 반복적 적용 (조합)

## 핵심 요약

### 기억해야 할 3가지

1. **혁신은 조합이다**: 완전히 새로운 것은 드물다. 기존 요소의 재조합이 혁신이다
2. **체계적으로 조합하라**: SCAMPER 7가지 질문으로 체계적 탐색
3. **인접부터 시작하라**: 너무 먼 도약은 실패. 한 걸음씩 인접 가능성 확장

### 조합의 수학

**n개 요소의 조합 폭발**:
- 10개 요소 → 45개 조합 (2개씩)
- 20개 요소 → 190개 조합 (2개씩)
- **통찰**: 요소를 늘리는 것만으로도 혁신 공간이 기하급수적 확장

### 즉시 실행 (20분)

**SCAMPER 빠른 실행**:
1. 현재 제품/서비스 1개 선택
2. SCAMPER 7가지 질문 각각에 2분씩 투자 (총 14분)
3. 21개 아이디어 중 가장 쉬운 것 1개 선택
4. 오늘 또는 이번 주에 프로토타입 테스트

**조합 매트릭스 (5분)**:
1. 우리가 가진 역량/자산 3개 나열
2. 외부에서 가져올 수 있는 요소 3개 나열
3. 3×3 = 9개 조합 생성
4. 가장 실현 가능한 조합 1개 선택, 1주일 실험
