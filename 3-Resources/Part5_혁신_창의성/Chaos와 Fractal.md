# Chaos와 Fractal

## 작은 차이가 만드는 패턴

### Chaos Theory (혼돈 이론)

초기 조건의 미세한 차이가 결과에 엄청난 차이를 만든다.

**핵심 개념:**
- 나비 효과(Butterfly Effect)
- 결정론적이지만 예측 불가능
- 민감한 초기 조건 의존성

**의미:**
- 작은 행동이 큰 변화를 만든다
- 시스템은 비선형적
- 통제보다 적응이 중요

### Fractal (프랙탈)

부분이 전체와 닮은 자기 유사성(Self-similarity) 구조.

**특징:**
- 스케일 불변성
- 무한한 복잡성
- 단순한 규칙의 반복

**자연의 예:**
- 해안선, 나뭇가지, 혈관
- 구름, 산맥
- 경제 데이터, 주가 패턴

### 패턴의 창발

복잡한 패턴은 단순한 규칙의 반복에서 나온다.

**통찰:**
- 복잡성 ≠ 복잡한 원인
- 반복과 피드백이 패턴을 만든다
- 부분에서 전체를 이해할 수 있다

### 연결

단순한 규칙의 반복으로 복잡한 패턴이 창발한다는 관점에서 [[TRIZ와 Isomorphism]]과 연결된다. 서로 다른 영역에서 동일한 구조가 반복된다는 동형성 개념은 프랙탈의 자기 유사성과 같은 원리다.
