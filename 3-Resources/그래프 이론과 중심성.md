# 그래프 이론과 중심성

병목 지점과 권한 위임

## 개요

그래프 이론의 중심성(Centrality) 개념을 조직 구조와 의사결정 흐름 분석에 활용.

## 중심성 지표

- **Degree Centrality**: 직접 연결 수
- **Betweenness Centrality**: 경로 사이에 위치하는 정도 (병목 지점)
- **Closeness Centrality**: 전체와의 평균 거리
- **Eigenvector Centrality**: 영향력 있는 노드와의 연결

## 적용

조직 내 병목 지점을 찾아내고, 효율적인 권한 위임과 커뮤니케이션 구조 설계에 활용.
