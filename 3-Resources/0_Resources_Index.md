## 사용 가이드

### 처음 방문하는 경우

1. **[[0_BrainTwin_Master_Index|BrainTwin 마스터 인덱스]]**에서 전체 프레임워크 이해
2. 관심 있는 Part의 인덱스 파일 읽기
3. 구체적인 개념 노트로 이동하여 학습

### 특정 문제를 해결하려는 경우

1. 위의 "문제 유형별 진입점"에서 해당 개념 찾기
2. 개념 노트 읽고 실전 적용 방법 확인
3. 관련 개념들을 함께 학습하여 통합적 이해

### 체계적으로 학습하려는 경우

1. Part 1 → Part 2 → Part 3 → Part 4 → Part 5 → Part 6 순서 추천
2. 각 Part의 인덱스 파일에서 학습 경로 참고
3. [[0_BrainTwin_Master_Index#학습 로드맵|학습 로드맵]] 활용
