1. **복잡성 단순화** - 본질만 남기고 제거
2. **데이터 기반** - 직관이 아닌 수학적 근거
3. **구조적 사고** - 증상이 아닌 근본 원인
4. **점진적 개선** - 완벽보다 학습
5. **실용주의** - 이론이 아닌 문제 해결
