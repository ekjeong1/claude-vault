## 의존성 그래프 (시각화)

### 주요 의존성 체인

#### 체인 1: 조직 분석

```
그래프 이론과 중심성 (Level 0)
    ↓
Rank와 Nullity (Level 1)
    ↓
Null Space 응용 (Level 2)
    ↓
최적화 이론 (Level 2)
```

**학습 경로:** 조직 병목 → 가치 분석 → 비효율 제거 → 자원 최적화
**총 소요 시간:** 3시간 10분
**실무 적용:** 조직 구조 개선

---

#### 체인 2: 확률적 의사결정

```
베이즈 정리 (Level 0)
    ↓
마르코프 체인 (Level 1)
    ↓
강화학습 (Level 3)
    ↓
AI와 Cognitive Twin (Level 5)
```

**학습 경로:** 믿음 업데이트 → 상태 전이 → 학습 → AI 자동화
**총 소요 시간:** 4시간 15분
**실무 적용:** 적응적 의사결정 시스템

---

#### 체인 3: 복잡성 관리

```
시스템 사고와 피드백 루프 (Level 0)
    ↓
Phase Transition (Level 1)
    ↓
Chaos와 Fractal (Level 4)
```

**학습 경로:** 피드백 구조 → 임계점 → 복잡계
**총 소요 시간:** 3시간 10분
**실무 적용:** 비선형 시스템 관리

---

#### 체인 4: 데이터 기반 실험

```
베이즈 정리 (Level 0)
    ↓
엔트로피 (Level 1)
    ↓
인과 추론 (Level 2)
    ↓
실험과 학습 (Level 3)
```

**학습 경로:** 불확실성 → 정량화 → 인과관계 → A/B 테스트
**총 소요 시간:** 3시간 55분
**실무 적용:** 실시간 최적화

---

#### 체인 5: 리스크 관리

```
엔트로피 (Level 1)
    ↓
포트폴리오 이론 (Level 2)
    ↓
극값 이론 (Level 3)
```

**학습 경로:** 불확실성 → 분산 → 극단 리스크
**총 소요 시간:** 2시간 35분
**실무 적용:** 포트폴리오 최적화
