# 전략과 경쟁 개념 모음

전략적 의사결정, 경쟁 상황, 불확실성 관리에 대한 핵심 이론들.

**↑ [[../0_BrainTwin_Master_Index|BrainTwin 마스터 인덱스로 돌아가기]]**

## 개요

이 폴더는 합리성과 비합리성, 균형과 전환, 리스크와 최적화, 네트워크 효과를 다루는 5가지 개념을 담고 있다. 각 개념은 독립적이지만 서로 연결되어 있다.

## 핵심 개념

### [[내쉬 균형]]
게임 이론의 핵심. 아무도 먼저 움직이지 않는 교착 상태와 그것을 깨는 메타 전략.

**핵심 질문**:
- 어떻게 안정적 균형을 만들 것인가?
- 어떻게 균형을 깨고 유리한 위치로 이동할 것인가?

### [[네트워크 효과와 플랫폼 경제학]]
사용자가 사용자를 부르는 기하급수적 성장. 임계 질량과 승자독식의 수학.

**핵심 질문**:
- 어떻게 임계 질량을 달성할 것인가?
- 네트워크 효과를 어떻게 방어하고 강화할 것인가?

### [[포트폴리오 이론]]
리스크와 수익의 최적화. 진정한 분산이란 무엇인가.

**핵심 질문**:
- 어떻게 리스크를 관리할 것인가?
- 집중과 분산의 균형을 어떻게 잡을 것인가?

### [[Phase Transition]]
임계점 돌파와 비선형 성장. 작은 변화가 거대한 전환을 만드는 순간.

**핵심 질문**:
- 언제 임계점이 오는가?
- 어떻게 tipping point를 만들고 넘을 것인가?

### [[행동경제학]]
인간의 실제 의사결정. Prospect Theory, Anchoring, 그리고 편향들.

**핵심 질문**:
- 사람들은 실제로 어떻게 행동하는가?
- 편향을 어떻게 활용하거나 극복할 것인가?

## 개념 간 연결

```
합리성 가정 ───────────→ 실제 행동
   ↓                        ↓
내쉬 균형              행동경제학
   ↓                        ↓
균형 깨기 ←──────────→ 편향 활용
   ↓                        ↓
Phase Transition  ←─→  군중 심리
   ↑                        ↑
임계점 타이밍 ←───────→ 리스크 관리
   ↑                        ↑
집중 전략              포트폴리오 이론
```

### 주요 연결 패턴

1. **합리성 vs 비합리성**
   - [[내쉬 균형]]: 합리적 행위자 가정
   - [[행동경제학]]: 실제 인간의 편향
   - 전략: 비합리성을 전략적으로 활용

2. **안정 vs 전환**
   - [[내쉬 균형]]: 안정적 교착 상태
   - [[Phase Transition]]: 균형을 깨고 급격한 전환
   - 전략: 언제 안정을 유지하고, 언제 깰 것인가

3. **집중 vs 분산**
   - [[포트폴리오 이론]]: 리스크 분산
   - [[Phase Transition]]: 임계점 돌파를 위한 집중
   - 전략: 타이밍에 따른 전환

4. **이론 vs 실제**
   - 이론적 최적: [[내쉬 균형]], [[포트폴리오 이론]]
   - 실제 행동: [[행동경제학]]
   - 현실 적용: 편향을 고려한 전략 설계

## 적용 영역

- **비즈니스**: 경쟁 전략, 가격 결정, 시장 진입
- **투자**: 포트폴리오 구성, 타이밍, 리스크 관리
- **성장**: 임계점 식별, 집중 투입, 스케일업
- **협상**: 균형 만들기, 프레이밍, 앵커링
- **조직**: 문화 변화, 의사결정 구조, 인센티브 설계

## 실전 질문

전략적 의사결정 시 스스로에게 물어볼 질문들:

1. 현재 상황은 어떤 균형 상태인가? ([[내쉬 균형]])
2. 네트워크 효과가 있는가? 임계 질량은? ([[네트워크 효과와 플랫폼 경제학]])
3. 리스크를 어떻게 분산하고 있는가? ([[포트폴리오 이론]])
4. 임계점은 어디이고, 언제 오는가? ([[Phase Transition]])
5. 사람들의 실제 행동 패턴은 무엇인가? ([[행동경제학]])

## Part 1 조직·인력과의 연결

Part 3의 전략 개념은 Part 1의 조직 분석에 직접 적용 가능하다:

### 균형과 병목
- **[[내쉬 균형]]**의 교착 상태 = **[[../Part1_조직_인력/그래프 이론과 중심성]]**의 조직 병목
- 조직의 권력 구조는 내쉬 균형이다: 누구도 먼저 권한을 포기하지 않는다
- 균형을 깨는 방법 = 병목 해소: 인센티브 재설계, 역할 재정의, 구조 개편

### 포트폴리오와 독립성
- **[[포트폴리오 이론]]**의 진정한 분산 = **[[../Part1_조직_인력/Rank와 Nullity]]**의 독립적 기여
- 팀들이 중복 업무를 한다 = 상관관계 높은 자산 (분산 효과 없음)
- 독립적 가치 창출 = 상관관계 낮은 자산 (실제 리스크 분산)

### 통합 프레임워크

조직 전략 설계에 두 파트를 함께 활용:

**1단계: 구조 분석** ([[../Part1_조직_인력/0_Part1_Organization_Index|Part 1]])
- [[../Part1_조직_인력/그래프 이론과 중심성|그래프 이론]]으로 병목 식별
- [[../Part1_조직_인력/Rank와 Nullity]]로 중복도 측정
- [[../Part1_조직_인력/군론과 대칭성|군론]]으로 반복 패턴 분석

**2단계: 전략 설계** (Part 3)
- 병목 = [[내쉬 균형]] → 어떻게 깰 것인가?
- 독립성 = [[포트폴리오 이론]] → 리스크를 어떻게 분산할 것인가?
- 변화의 타이밍 = [[Phase Transition]] → 임계점은 언제인가?

**실전 질문**:
- 현재 조직 구조가 만드는 교착 상태(균형)는 무엇인가?
- 팀 간 독립성이 실제 리스크 분산을 만드는가, 아니면 겉만 나눈 것인가?
- 조직 변화의 임계점은 어디이고, 언제 집중 투입해야 하는가?
