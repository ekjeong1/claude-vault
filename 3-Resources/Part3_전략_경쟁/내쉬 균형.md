# 내쉬 균형

## 정의

각 참여자가 다른 참여자의 전략을 고정으로 간주할 때, 자신의 전략을 바꿔도 더 나은 결과를 얻을 수 없는 상태.

## 핵심 개념

- **교착 상태**: 누구도 먼저 움직이지 않는 균형점
- **안정성**: 일방적 이탈이 불리한 구조
- **예측 가능성**: 합리적 행위자들의 수렴점

## 메타 전략

### 균형 깨기
- **게임 구조 변경**: 규칙, 보상 체계, 참여자 구성 자체를 바꾸기
- **비대칭 정보 활용**: 상대가 모르는 정보로 먼저 움직이기
- **외부 충격 만들기**: 새로운 기술, 규제, 시장 진입으로 균형 붕괴

### 협력적 균형 유도
- **반복 게임**: 장기적 관계에서 협력이 내쉬 균형이 됨 (죄수의 딜레마 → Tit-for-Tat)
- **신뢰 신호**: 선제적 협력, 투명성, 인센티브 정렬
- **구속 장치**: 계약, 담보, 평판 시스템으로 이탈 비용 증가

### 다중 균형 상황
- **초기 조건의 힘**: 여러 균형이 가능할 때 먼저 도달한 균형이 고착화
- **조정 게임**: 모두가 같은 선택을 하면 유리 (키보드 배열, 운전 방향)
- **선점 전략**: 먼저 움직여서 유리한 균형으로 유도

### 전략적 비합리성
- **commitment device**: 합리적 선택지를 의도적으로 제거 (다리 끊기)
- **편향 활용**: [[행동경제학]]의 휴리스틱을 전략적으로 이용
- **불확실성 조성**: 예측 불가능성으로 상대방 전략 무력화

## 사례

### 고전 사례
- 과점 시장의 가격 경쟁
- 군비 경쟁
- 표준 전쟁

### 현대 적용 사례

**소셜 미디어 플랫폼**
- 모든 플랫폼이 유사한 기능(스토리, 릴스, 숏폼)을 복제하는 균형
- 차별화 시도 → 사용자 이탈 리스크 → 모방 전략으로 수렴
- 균형을 깨는 방법: 완전히 새로운 사용자층 공략 (틱톡의 Z세대, 비핸드의 B2B)

**채용 시장의 학벌 신호**
- 기업들이 학벌로 필터링 → 학생들이 명문대 집착 → 기업이 계속 학벌 활용
- 악순환적 균형: 개별 행위자가 이탈하면 불리
- 균형 깨기 시도: 블라인드 채용, 포트폴리오 기반 평가

**암호화폐 네트워크 효과**
- 비트코인 우위 = 내쉬 균형 (가장 많은 사용자 → 가장 안전 → 더 많은 사용자)
- 새로운 코인이 기술적으로 우수해도 균형 깨기 어려움
- 성공 전략: 틈새 시장(이더리움의 스마트 컨트랙트)으로 별도 균형 창출

## 통찰과 함정

### 균형의 역설
- **효율성 ≠ 균형**: 파레토 최적이 아닌 균형도 안정적 (죄수의 딜레마)
- **관성의 힘**: 나쁜 균형도 깨기 어려움 (QWERTY 키보드, VHS vs Betamax)
- **다수의 균형**: 같은 게임에 여러 균형 존재 가능 (어느 균형에 도달할지는 역사 의존적)

### 실전 적용시 주의점
- **합리성 과신**: 실제 사람들은 완벽히 합리적이지 않음 → [[행동경제학]]
- **정보 비대칭**: 이론은 완전 정보 가정, 현실은 불완전
- **동적 vs 정적**: 내쉬 균형은 정적 개념, 현실은 계속 변화
- **진화적 균형**: 생물학적 비유 - 최적이 아니어도 생존 가능한 전략

### 메타 인사이트
- **균형 인식 = 전략적 기회**: 현재 균형을 보면 어디서 경쟁이 치열한지 알 수 있음
- **균형 깨기의 비용**: 단기 손해를 감수하고 장기 이득을 노려야 함
- **균형의 전환 비용**: 새로운 균형으로 이동하는 조정 비용 고려 필요
