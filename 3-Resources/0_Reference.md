## 📚 용어 사전

### A-C

**A/B 테스트 (A/B Test)**
두 버전을 비교하여 더 나은 것을 선택하는 실험 방법
→ [[Part4_의사결정_데이터/실험과 학습]]

**Betweenness Centrality**
네트워크에서 최단 경로가 가장 많이 지나가는 노드. 병목 지표.
→ [[Part1_조직_인력/그래프 이론과 중심성]]

**CVaR (Conditional Value at Risk)**
최악 X% 상황의 평균 손실
→ [[Part4_의사결정_데이터/극값 이론]]

---

### D-F

**동적 계획법 (Dynamic Programming)**
큰 문제를 작은 하위 문제로 나누어 최적 해 찾기
→ [[Part4_의사결정_데이터/동적 계획법과 강화학습]]

**엔트로피 (Entropy)**
불확실성의 정량적 측정. 높을수록 불확실.
→ [[Part4_의사결정_데이터/엔트로피]]

**Fractal**
자기 유사성을 가진 구조. 부분이 전체와 비슷.
→ [[Part5_혁신_창의성/Chaos와 Fractal]]

---

### G-M

**그래프 이론 (Graph Theory)**
노드와 엣지로 관계를 표현하고 분석하는 수학
→ [[Part1_조직_인력/그래프 이론과 중심성]]

**내쉬 균형 (Nash Equilibrium)**
아무도 일방적으로 전략을 바꾸지 않는 안정 상태
→ [[Part3_전략_경쟁/내쉬 균형]]

**Null Space**
결과에 기여하지 않는 벡터 공간. 비효율 프로세스 식별.
→ [[Part2_프로세스_운영/Null Space 응용]]

**마르코프 체인 (Markov Chain)**
이전 상태만 고려하는 확률적 상태 전이 모델
→ [[Part2_프로세스_운영/마르코프 체인]]

**Multi-Armed Bandit**
탐색과 활용의 균형. 실시간 최적화.
→ [[Part4_의사결정_데이터/실험과 학습]]

---

### P-R

**PCA (Principal Component Analysis)**
주성분 분석. 고차원 데이터를 저차원으로 압축.
→ [[Part2_프로세스_운영/SVD와 PCA]]

**Phase Transition**
임계점에서의 급격한 상태 변화
→ [[Part3_전략_경쟁/Phase Transition]]

**Rank**
행렬의 독립적인 행/열의 개수. 실질적 차원.
→ [[Part1_조직_인력/Rank와 Nullity]]

**강화학습 (Reinforcement Learning)**
시행착오를 통해 최적 정책을 학습하는 방법
→ [[Part4_의사결정_데이터/동적 계획법과 강화학습]]

---

### S-Z

**SVD (Singular Value Decomposition)**
특이값 분해. 행렬을 3개 행렬의 곱으로 분해.
→ [[Part2_프로세스_운영/SVD와 PCA]]

**시스템 사고 (Systems Thinking)**
요소 간 상호작용과 피드백 루프를 고려하는 사고방식
→ [[Part2_프로세스_운영/시스템 사고와 피드백 루프]]

**TRIZ**
발명적 문제 해결 이론. 40가지 혁신 원리.
→ [[Part5_혁신_창의성/TRIZ와 Isomorphism]]

**VaR (Value at Risk)**
X% 신뢰 수준에서 최대 예상 손실
→ [[Part4_의사결정_데이터/극값 이론]]

**베이즈 정리 (Bayes' Theorem)**
새로운 증거로 믿음을 업데이트하는 확률 공식
→ [[Part2_프로세스_운영/베이즈 정리]]

**행동경제학 (Behavioral Economics)**
실제 인간의 비합리적 의사결정 연구
→ [[Part3_전략_경쟁/행동경제학]]

## 🗺️ 개념 지도

### Part 간 연결

```
Part 1 (조직) ←→ Part 3 (전략)
- 병목 = 내쉬 균형의 교착 상태
- 독립성 = 포트폴리오의 진정한 분산

Part 2 (프로세스) ←→ Part 4 (의사결정)
- SVD/PCA ←→ 엔트로피 (정보 압축)
- 마르코프 ←→ 동적 계획법 (상태 전이)
- 베이즈 ←→ 실험과 학습 (점진적 학습)

Part 5 (혁신) ←→ Part 6 (미래)
- TRIZ ←→ AI (패턴 인식 자동화)
- Fractal ←→ Quantum (스케일 불변 최적화)
```

---

### 개념 간 시너지

**병목 제거 + 프로세스 최적화:**
- 그래프 이론 (병목 식별) + Null Space (비효율 제거)

**데이터 기반 의사결정:**
- 인과 추론 (원인) + 베이즈 (업데이트) + Multi-Armed Bandit (실시간)

**리스크 관리:**
- 포트폴리오 이론 (분산) + 극값 이론 (극단)

**전략 수립:**
- 내쉬 균형 (경쟁) + 행동경제학 (실제) + Phase Transition (타이밍)

## 🔗 의존성 그래프

### 학습 순서 추천

**기초 개념 (먼저 학습):**
```
그래프 이론
     ↓
Rank-Nullity
     ↓
SVD/PCA
```

**중급 개념 (기초 후):**
```
마르코프 체인 → 동적 계획법
베이즈 정리 → 실험과 학습
내쉬 균형 → Phase Transition
```

**고급 개념 (중급 후):**
```
시스템 사고 (기초 + 중급 통합)
포트폴리오 이론 (중급 필요)
극값 이론 (확률론 이해 필요)
```
