# Null Space 응용

비효율 프로세스를 발견하고 제거하는 선형대수 관점.

## 핵심 개념

**Null Space**(영공간)는 행렬에 곱해도 0이 되는 벡터들의 집합. "아무 영향도 주지 못하는 방향"을 의미.

## 비즈니스 적용

- 업무 중 결과에 기여하지 않는 활동 식별
- 투입 대비 산출이 0인 프로세스 제거
- 조직 내 불필요한 보고·회의 최소화

## 예시

10개 활동을 행렬로 모델링 → null space 분석 결과 3개 활동이 성과에 영향 없음 → 해당 활동 중단

## 효과

자원 낭비 제거, 팀 생산성 향상, 핵심 업무 집중
