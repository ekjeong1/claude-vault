# 수학적 의사결정 도구

비즈니스 복잡성을 단순화하고 데이터 기반 의사결정을 지원하는 수학·통계 개념 모음.

**↑ [[../0_BrainTwin_Master_Index|BrainTwin 마스터 인덱스로 돌아가기]]**

## 개요

이 4개 노트는 모두 **복잡한 현실을 단순화**하고 **불필요한 것을 제거**하며 **핵심에 집중**하는 공통 철학을 공유한다.

- **선형대수** (SVD/PCA, Null Space) - 차원 축소, 불필요 제거
- **확률론** (마르코프 체인, 베이즈 정리) - 불확실성 모델링, 학습

## 노트 목록

### 선형대수 기반

**[[SVD와 PCA]]**
복잡한 KPI를 단순화하는 차원 축소 기법. 수십 개 지표를 2-3개 핵심 축으로 압축.

**[[Null Space 응용]]**
비효율 프로세스 제거. 결과에 기여하지 않는 활동을 수학적으로 식별.

### 확률론 기반

**[[마르코프 체인]]**
고객 행동의 전이 확률 모델링. 상태 간 이동 패턴으로 LTV 예측.

**[[베이즈 정리]]**
데이터에 따른 전략 업데이트. 새로운 증거로 믿음을 점진적으로 갱신.

## 활용 시나리오

### 의사결정 단순화
1. [[SVD와 PCA]]로 복잡한 대시보드를 핵심 지표로 압축
2. [[Null Space 응용]]으로 무의미한 활동 제거
3. [[베이즈 정리]]로 단순화된 지표 기반 전략 업데이트

### 고객 행동 분석
1. [[마르코프 체인]]으로 고객 여정 모델링
2. [[베이즈 정리]]로 전이 확률을 실시간 데이터로 갱신
3. [[SVD와 PCA]]로 고객 세그먼트를 핵심 차원으로 단순화

### 프로세스 최적화
1. [[Null Space 응용]]으로 비효율 업무 식별
2. [[SVD와 PCA]]로 성과 지표의 진짜 동인 파악
3. [[마르코프 체인]]으로 프로세스 상태 전환 최적화

## 공통 원칙

- **복잡성 단순화**: 본질만 남기고 나머지는 제거
- **데이터 기반**: 직관이 아닌 수학적 근거
- **점진적 개선**: 완벽한 모델이 아닌 학습하는 시스템
- **실용주의**: 이론이 아닌 비즈니스 문제 해결
